# Chip-8 Memory Map

| Memory Range | Description                                |
|--------------|--------------------------------------------|
| 0x000 - 0x1FF| Reserved for interpreter                   |
| 0x200 - 0xFFF| Chip-8 Program/Data Space                  |
| 0x200 - 0x5FF| Start of most Chip-8 programs              |
| 0x600 - 0xFFF| Start of ETI 660 Chip-8 programs           |
